# Project 01: Indian Census Simulation Survey Form

A strictly validated, semantically built HTML5 form designed to satisfy 100% of the freeCodeCamp Full-Stack certification test criteria.

## 🎯 Project Objective
The objective was to transition out of sandbox environments to implement complex, structural HTML inputs locally using VS Code and Live Server.

## 🛠️ Context & Features
Instead of standard tutorial templates, this project models an official **Government of India Census Survey** tracking population demographic shifts since 2011.

- **Comprehensive Data Fields**: Features exhaustive demographic data inputs including an engineered 36-option Indian State/UT selection dropdown.
- **Strict Validations**: Applies explicit range limits on age criteria constraints (`min="1"`/`max="100"`), alongside critical mandate identifiers (`required`).
- **Form Control Grouping**: Implements robust field separation for conditional metrics (logical radio button scoping and multi-selection checkbox arrays).

## 🧠 Key Learnings
- Resolved strict test suite architecture parameters (ID matching, structural hierarchy).
- Mastered complex list populations within dropdown interfaces.
- Maintained clean, modern HTML5 syntax compliance without relying on external style frameworks.

## 💻 Local Setup
1. Open this workspace folder inside VS Code.
2. Ensure the **Live Server** extension is active.
3. Launch and view the build locally on port `5500`.
