# 📚 Bespoke Dark Premium Book Inventory Archive

A sophisticated, data-dense digital library archive panel built using semantic HTML5 and advanced CSS3 attribute selector engines. This project marks the successful completion of **Certification Project #3** in the freeCodeCamp Responsive Web Design curriculum, refactored from a basic passing layout into a luxury dark-mode dashboard.

## 🛠️ Architecture & Layout Mechanics

*   **Rigid Sizing Layout Engine**: Engineered via `table-layout: fixed;` to bypass default content-based browser calculation engines. Distributes the wide 1200px widescreen canvas real estate across mathematically precise columns (`35%`, `25%`, `15%`, `13%`, `12%`).
*   **Decoupled Class Selectors**: Implemented standalone word-attribute matching operators (`~=`) and prefix anchors (`^=`) to cleanly isolate and style target structures (`.status` tags and `.rate` nodes) across different reading condition categories.
*   **Hardware-Accelerated Micro-Interactions**: Leverages fluid `translateY` transform rules combined with high-performance filter matrices (`filter: brightness(1.2)`) to provide smooth visual elevation and feedback as users mouse over the records.

## 🎨 Design System Specifications

*   **Typography Framework**: High-end editorial pairing features **Cinzel** for a commanding, widely-spaced roman header system, backed by **Cormorant Garamond** for luxury serif row data readability.
*   **The Theme**: Built on a minimalist Obsidian canvas (`#0b0f17`) featuring custom `135deg` structural diagonal gradients for row blocks, 4px boundary accent lines for reading states, and glowing rating indicator elements (`box-shadow: 0 0 8px`).
